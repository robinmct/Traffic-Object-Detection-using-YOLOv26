# `YOLOv26: Vehicle Detection in Traffic Footage`

## Project Overview
This repository contains a machine learning pipeline for detecting vehicles in traffic video footage. It features a custom video derived dataset and utilizes a YOLO object detection model to identify vehicles in complex traffic environments.

## Key Features
* **Custom Dataset Generation:** Built by extracting and manually annotating frames directly from raw traffic video footage.
* **YOLO Object Detection:** Trained to accurately draw bounding boxes around various vehicles.
* **Complete Pipeline:** Covers data extraction, annotation, model training, and video inference.

## Tech Stack
* **Computer Vision:** YOLO, OpenCV
* **Language:** Python
* **Data Handling:** NumPy, Pandas
